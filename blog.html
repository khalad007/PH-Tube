# Null & Undefiend

Null

Null in JavaScript means an empty value and is also a primitive type in JavaScript.

Undefiend

8 Ways to get Undefiend

Undefined, means the variable has been declared, but its value has not been assigned.

1. Variable that is not initialized
2. function with no return
3. Parameter that is not passed will be Undefiend
4. If return has nothing on the right side will return Undefiend
5. Property that doesn't exist on an object will give you Undefiend
6. Accessing array elements outside of the index range
7. deleting an element inside an array
8. Set a value directly to Undefiend
